## Goal

When a pasted reference's URL carries a tracking trace left by an AI tool (for example `https://qtfn.org.au/?utm_source=chatgpt.com`), surface a clear warning so the user knows the citation was likely copied straight out of an AI assistant rather than found in a real source.

## What gets flagged

A URL is treated as an AI trace if it contains any known AI-assistant marker in its query/parameters, case-insensitive:

```text
utm_source=chatgpt.com   utm_source=chatgpt    utm_source=openai
utm_source=perplexity    utm_source=perplexity.ai
utm_source=gemini        utm_source=bard       utm_source=google-gemini
utm_source=claude        utm_source=anthropic
utm_source=copilot       utm_source=bingchat   utm_source=you.com
utm_source=poe.com
```

Also catch the same names appearing as `ref=`, `source=`, or `utm_medium=` values. Matching is substring-based on the lowercased URL so future param shapes still trip it.

## Changes

### 1. `src/lib/reference-check.functions.ts`
- Add an `aiTrace: string` field to the `ReferenceResult` interface (empty string when none; the matched marker text when found, e.g. `"utm_source=chatgpt.com"`).
- Add a small helper `detectAiTrace(ref: string): string` that scans the full reference text (not just the extracted URL, in case the trailing param got split) against the marker list and returns the first match or `""`.
- In `processReference`, call it once and set `base.aiTrace`. Append a sentence to `base.notes` when a trace is found (e.g. `"AI-tool tracking parameter detected (utm_source=chatgpt.com) — this citation was likely generated by an AI assistant."`).
- Include `aiTrace: ""` in the error fallback object in `runPool` so the type stays satisfied.
- Leave the existing verdict logic intact — the AI trace is an independent warning shown alongside the normal verdict, not a replacement.

### 2. `src/components/ReferenceResultCard.tsx`
- When `result.aiTrace` is set, render a distinct warning badge next to the verdict/type badges (amber/red styling, `Sparkles` or `Bot` icon from lucide-react, label like `AI trace`).
- Show the matched marker in the details list (a new `dt/dd` row: `AI trace: utm_source=chatgpt.com`) so the user sees exactly what triggered it.
